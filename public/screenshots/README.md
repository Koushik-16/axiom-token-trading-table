# Screenshots

This directory contains screenshots of the application for documentation purposes.

## Required Screenshots

1. **desktop.png** - Full desktop view of the token table
2. **mobile.png** - Mobile responsive view (320px - 640px)
3. **realtime.mp4** - Real-time price update animation
4. **modal.png** - Token details modal
5. **loading.png** - Loading skeleton states
6. **tabs.png** - All three tab views (New Pairs, Final Stretch, Migrated)

## How to Capture

### Desktop View
1. Open app at http://localhost:3000
2. Set browser width to 1920px
3. Capture full page screenshot

### Mobile View
1. Open Chrome DevTools
2. Toggle device toolbar (Cmd+Shift+M / Ctrl+Shift+M)
3. Select device or custom (320px width)
4. Capture screenshot

### Feature Screenshots
1. Interact with specific features
2. Capture at the right moment
3. Annotate if necessary

## Tools

- **macOS**: Cmd + Shift + 4
- **Windows**: Windows + Shift + S
- **Chrome**: DevTools > Device Toolbar > Screenshot icon
- **Firefox**: Right-click > Take Screenshot
